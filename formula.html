Consolidated Formula List for Mechanical Engineering Exam

Prepared for the Nepal Public Service Commission’s Mechanical Engineering Service, General Mechanical Subgroup, Non-Gazetted First Class exam. Covers all formula-based topics from the syllabus.

Applied Mechanics

Statics

  • ΣF = 0 (equilibrium: net force is zero)
  • F = ma (force = mass × acceleration)
  • F_action = -F_reaction (Newton’s third law)
  • M = F × d (moment of a force: force × perpendicular distance)
  • F_friction = μN (friction force: μ = coefficient of friction, N = normal force)
  • μ = tanθ (coefficient of friction: θ = angle of friction or repose)
  • x_cg = (Σm_i x_i) / Σm_i (center of gravity for composite body: weighted average of centroids)
  • J = (πr^4)/2 (polar moment of inertia, circular shaft)
  • τ = (T × r) / J (shear stress in shaft: T = torque, r = radius, J = polar moment)

Kinematics

  • s = ut + (1/2)at² (distance traversed, uniform acceleration: u = initial velocity, a = acceleration, t = time)
  • v = u + at (velocity)
  • s = vt (distance, constant velocity)
  • R_max = (u²/g) at θ = 45° (projectile motion, max range: u = initial velocity, g = 9.8 m/s²)

Dynamics

  • W = Fd cosθ (work: θ = angle between force and displacement)
  • KE = (1/2)mv² (kinetic energy)
  • PE = mgh (potential energy: h = height)
  • P = W/t or P = Fv (power: v = velocity)
  • T = F × r or T = Iα (torque: I = moment of inertia, α = angular acceleration)
  • I = (1/4)MR² (moment of inertia, circular disc, about diameter)
  • P = Tω (power from torque: ω = angular velocity, rad/s)
  • m₁u₁ + m₂u₂ = m₁v₁ + m₂v₂ (conservation of momentum, isolated system)
  • Total energy (KE + PE) = constant (conservation of energy)
  • δ_max = (WL^3)/(48EI) (maximum deflection, simply supported beam, central load: W = load, L = length, E = Young’s modulus, I = moment of inertia)

Thermodynamics

Gas Laws

  • PV = nRT (ideal gas law: P = pressure, V = volume, n = moles, R = 8.314 J/mol.K, T = Kelvin)
  • P₁V₁ = P₂V₂ (Boyle’s law, constant T)
  • V₁/T₁ = V₂/T₂ (Charles’ law, constant P)
  • (P₁V₁)/T₁ = (P₂V₂)/T₂ (combined gas law)

First Law

  • ΔU = Q - W (first law: ΔU = change in internal energy, Q = heat added, W = work done by system)
  • ΔU = nC_vΔT (internal energy, ideal gas: C_v = specific heat at constant volume)
  • W = PΔV (work done, constant pressure)

Second Law

  • η = 1 - (T_cold/T_hot) (Carnot efficiency, T in Kelvin)
  • ΔS ≥ 0 (entropy change, isolated system: S = entropy)
  • η = W/Q_in (thermal efficiency: W = work output, Q_in = heat input)

Processes

  • Isothermal: W = nRT ln(V₂/V₁), ΔU = 0 (constant T)
  • Adiabatic: PV^γ = constant, W = C_v (T₂ - T₁), T₁V₁^(γ-1) = T₂V₂^(γ-1) (no heat transfer, γ = C_p/C_v, e.g., 1.4 for diatomic gas)
  • Isochoric: W = 0, ΔU = Q = nC_vΔT (constant V)
  • Isobaric: W = P (V₂ - V₁), Q = nC_pΔT (constant P: C_p = specific heat at constant pressure)
  • Polytropic: PV^n = constant, W = (P₂V₂ - P₁V₁)/(1 - n)

Cycles

  • Otto: η = 1 - (1/r^(γ-1)) (r = compression ratio)
  • Diesel: η = 1 - (1/r^(γ-1)) × [(r_c^γ - 1)/(γ(r_c - 1))] (r_c = cut-off ratio)

Additional

  • C_p - C_v = R (relationship between specific heats)

Heat Engines

Cycle Efficiencies

  • Otto: η = 1 - (1/r^(γ-1)) (r = compression ratio, γ = adiabatic index, e.g., 1.4 for air)
  • Diesel: η = 1 - (1/r^(γ-1)) × [(r_c^γ - 1)/(γ(r_c - 1))] (r_c = cut-off ratio)
  • Brayton: η = 1 - (1/r_p^((γ-1)/γ)) (r_p = pressure ratio)
  • r = V_total / V_clearance (compression ratio)
  • r_c = V_cut-off / V_clearance (cut-off ratio, Diesel)

Work and Power

  • W = Q_in - Q_out (work output: Q_in = heat added, Q_out = heat rejected)
  • P = W/t or P = Tω (power output: T = torque, ω = angular velocity, rad/s)
  • IP = (P_m × L × A × N × n)/(60) (indicated power: P_m = mean effective pressure, L = stroke length, A = piston area, N = RPM, n = number of cylinders; for 4-stroke, divide by 2)
  • BP = (2πNT)/60 (brake power: N = RPM, T = brake torque)
  • MEP = W_net / V_d (mean effective pressure: V_d = displacement volume)

Efficiency Metrics

  • η_mech = BP/IP (mechanical efficiency)
  • η_th = W/Q_in (thermal efficiency)
  • SFC = Fuel consumption (kg/h) / BP (kW) (specific fuel consumption)

Electro-Mechanical

  • P = VI cosφ (electrical power: V = voltage, I = current, cosφ = power factor)
  • P_mech = Tω (mechanical power, motor: T = torque, ω = angular velocity)
  • η = (P_out / P_in) × 100 (motor efficiency: P_out = P_mech, P_in = P_elec)
  • η = (P_elec / P_mech) × 100 (generator efficiency)
  • T = (P × 60) / (2πN) (torque: P = power in watts, N = RPM)
  • E_b = V - I_a R_a (back EMF, DC motor: V = applied voltage, I_a = armature current, R_a = armature resistance)
  • N_s = (120f) / p (synchronous speed, AC motor: f = frequency in Hz, p = number of poles)
  • P_cu = I²R (copper loss: I = current, R = resistance)
  • s = (N_s - N_r) / N_s (slip, induction motor: N_s = synchronous speed, N_r = rotor speed)
  • C = (P (tanφ₁ - tanφ₂)) / (2πf V²) (power factor correction: C = capacitance, P = power, φ₁/φ₂ = initial/final phase angles)

Industrial Boiler

  • η_b = (m_s (h_s - h_f)) / (m_f × CV) (thermal efficiency: m_s = steam mass, h_s = steam enthalpy, h_f = feedwater enthalpy, m_f = fuel mass, CV = calorific value)
  • η_overall = (Steam energy output / Fuel energy input) × 100 (overall efficiency)
  • Q = m_f × CV (heat supplied)
  • Q_loss = m_g × C_p × (T_g - T_a) (heat loss, flue gas: m_g = flue gas mass, C_p = specific heat, T_g = flue gas temperature, T_a = ambient temperature)
  • E = (m_s (h_s - h_f)) / 2257 (equivalent evaporation: 2257 kJ/kg = latent heat of steam at 100°C, 1 atm)
  • Capacity = m_s / t (boiler capacity: kg/s or kg/h)
  • ΔP = h × ρ × g (draft pressure: h = chimney height, ρ = air density, g = 9.8 m/s²)
  • m_f = (m_s (h_s - h_f)) / (η_b × CV) (fuel consumption rate)

Estimating and Costing

  • Q_be = FC / (P - VC) (break-even point, units: FC = fixed cost, P = selling price per unit, VC = variable cost per unit)
  • S_be = FC / (1 - (VC/P)) (break-even sales, value)
  • ROI = (Net Profit / Investment) × 100 (return on investment)
  • NP = TR - TC (net profit: TR = total revenue, TC = total cost)
  • D = (C - S) / n (straight-line depreciation: C = initial cost, S = salvage value, n = useful life in years)
  • R = D / C × 100 (annual depreciation rate)
  • TC = FC + VC × Q (total cost: Q = quantity produced)
  • WC = Current Assets - Current Liabilities (working capital)
  • A = P / [(1 - (1 + r)^(-n)) / r] (amortization: P = principal, r = interest rate, n = periods)
  • CM = P - VC (contribution margin)
  • PM = (NP / TR) × 100 (profit margin)

Machine Drawing

  • T = Upper Limit - Lower Limit (tolerance range)
  • C = Hole Dimension - Shaft Dimension (fit clearance, for clearance fit)
  • A = πr² (area of circle, for isometric views)
